# ☁️ Proyecto Intermodular – Fundamentos de Computación en la Nube (MPO)

Este documento forma parte del Proyecto Intermodular del ciclo **Sistemas Microinformáticos y Redes (SMR)**.  
En este módulo se analiza y diseña una arquitectura cloud básica para el proyecto principal, utilizando **Microsoft Azure** como proveedor.

---

## 1. Elección del proveedor Cloud

**Proveedor elegido:** Microsoft Azure  
**Motivo:** Azure ofrece una integración perfecta con entornos Windows, Active Directory y VirtualBox, lo que facilita el despliegue del sistema operativo Windows 11 Pro utilizado en el proyecto.
